🧠 About the Project

This is a simple personal portfolio landing page designed to present my work, skills, and contact information cleanly and minimally.

The goal of the project was to:

- practice building UI without frameworks
- improve layout and styling skills using Tailwind CSS
- set up a fast, modern development workflow with Vite

  
🛠️ Tech Stack
- HTML5
- Vanilla JavaScript (ES6+)
- Tailwind CSS
- Vite

  
⚙️ Features
- Responsive layout (mobile / tablet / desktop)
- Smooth scrolling navigation
- Minimal and clean UI design
- Fast build and dev environment via Vite
- Optimized static landing page structure

🧑‍💻 Getting Started

To run the project locally:
```
git clone https://github.com/your-username/asyaspace.git
cd asyaspace
npm install
npm run dev
```

To build for production:
```
npm run build
```
